2 matches confirmed for Wrestlemania 32

2 matches have been confirmed for Wrestlemania from last week's RAW. The first is a No Holds Barred Street Fight bout between Dean Ambrose and Brock Lesnar. Personally, it seems to be like a fun match, however it would be interesting to see the dynamics between them.

The second match that was confirmed is Shane Mcmahon vs the Undertaker in a Hell in a Cell match, for the control of Monday Night RAW. It seems to be an interesting match, but since it is a Hell in a Cell, they risk getting Shane or the Undertaker injured.
